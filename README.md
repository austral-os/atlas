# Atlas Kernel

Atlas is the official Linux kernel for Austral OS.

**Atlas is not a heavily patched Linux fork.**

Its primary goal is to provide a carefully curated Linux kernel configuration for Austral OS while staying close to upstream Linux and Debian. This ensures maximum stability, compatibility, and long-term maintainability without the overhead of tracking and rebasing experimental out-of-tree patches.

## Project Structure

*   `configs/`: Contains the base Debian configuration and the derived Atlas configuration.
*   `docs/`: Project documentation, including the guiding principles.
*   `scripts/`: Automation scripts for building the kernel.
*   `ROADMAP.md`: The development roadmap for Atlas.
